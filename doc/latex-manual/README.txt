The LaTeX documentation source code for FLEXPART version 9.3.X is based on 
the existing  version 9.2 documentation provided by Nina Iren Kristiansen 
(NILU, Nina.Iren.Kristiansen@nilu.no). 

The source file (flexpart9.3.tex) is provided with the FLEXPARTv9.3.0 
distribution under doc/ so that any modification in the code by the model 
developers can as well be included in the corresponding documentation. 

The documentation source code can be compiled using integrated LaTeX 
environments suche as Kile (http://kile.sourceforge.net/) or  
Texmaker(http://www.xm1math.net/texmaker/) or via command line with:

latex -interaction=nonstopmode flexpart9.3.tex

to generate the dvi output or

pdflatex flexpart9.3.tex

to generate a pdf.
